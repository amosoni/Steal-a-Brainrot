🔍 诊断信息

如果游戏无法加载，可能的原因：

  • X-Frame-Options: 游戏网站设置了 DENY 或 SAMEORIGIN
  • CSP (Content Security Policy): 禁止了frame-ancestors
  • JavaScript 检测: 游戏检测到被嵌入而拒绝加载
  • 网络限制: 游戏网站限制了跨域访问

解决方案：

  • 联系游戏开发者，请求允许嵌入
  • 使用游戏提供的官方嵌入代码
  • 考虑使用游戏API而不是直接嵌入
  • 提供直接链接让用户在新窗口打开
